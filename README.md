# Simple Search Engine (C++)

## Overview
A high-performance command-line tool that indexes text files and allows for O(1) time complexity search queries using Hash Maps.

## Features
* **Inverted Indexing:** Maps words to file locations for instant lookup.
* **File Handling:** Recursively reads local text files.
* **Clean Interface:** Simple CLI commands for user interaction.

## Tech Stack
* C++ (STL: Maps, Vectors, Sets)
* File I/O
